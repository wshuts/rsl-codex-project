# HH Champion Relic Recommendation Inversion

## Scope

This dataset contains only champion-specific relic recommendations observed from HellHades champion relic recommendation screens reviewed in this chat.

Rules:
- One row per unique relic.
- Champion list contains only champions directly observed in HH recommendation screens.
- No interpretation or roster matching.
- Generic HH relic-page examples are excluded.
- Champions with no recommendations are tracked separately.
